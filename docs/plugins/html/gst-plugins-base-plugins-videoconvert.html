videoconvert

videoconvert — Converts video from one colorspace to another

Synopsis

Properties

Description

Convert video frames between a great variety of video formats.

Example launch line

gst-launch -v videotestsrc ! video/x-raw,format=\(fourcc\)YUY2 ! videoconvert ! ximagesink
